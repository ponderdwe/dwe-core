# ADR-003: Governance Model and Copyright for DWE-Core

## Status

- **Date:** 2026-02-27
- **Status:** Proposed

## Context

Ponder is making DWE-Core available as an open source project, with an open source licence and source code available on github. The goal of this project is to make data infrastructure more available to school networks, and to be open to collaboration with school networks or other ed-tech organizations in pursuit of that goal. Ponder is currently a small organization with only a few employees.

## Decision

At the beginning of this project, Ponder will own the copyright for the project, and be responsible for the governance of the project. Ponder employees will be responsible for decisions about the project, including accepting contributions from other parties. 

Ponder will publish the source for DWE-core to github. Ponder will track issues in github and publish roadmaps and project tracking there. Ponder employees will accept Pull Requests from possible contributors, using standard github mechanisms. Ponder will strive to be transparent about decision making, creating “Architectural Decision Records” (like this one) to capture reasoning.

Ponder may revisit this decision in the future if the project gains adoption by school networks and gains traction with contributions / collaborators who are not members of Ponder. 

## Consequences

As the project is in the early stages, we want to reduce overhead while being open to collaboration. Ponder employees are currently doing the work with minimal overhead; this governance structure allows Ponder to move rapidly to meet the needs of the initial schools adopting DWE while sharing transparently and being open to contributions.

The downside of corporate governance structures generally is a potential barrier to potential future contributors feeling welcome or motivated to contribute. We believe that in the initial stages, contributors are likely to be parties that we have a direct relationship with in our work directly with school networks. If this project gets traction, we anticipate revisiting this model.
